## What is this?

Clawible lets you build web apps by chatting with AI agents. Unlike cloud-based alternatives (Lovable, Manus, Replit, Bolt), everything runs locally:

- *Your code stays on your machine* (no cloud lock-in)
- *You pick the stack* (React, Next.js, .NET, Django, FastAPI, etc.)
- *You pick the database* (SQLite, Postgres, MySQL, MongoDB, etc.)
- *You pick the cloud (Azure, AWS, Google, Oracle, Netlify, Vercel, etc.)
- *You pick the AI model* (via OpenClaw - Claude, GPT, Gemini, local models)
